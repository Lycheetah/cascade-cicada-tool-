# CASCADE CICADA TOOL — Example Pyramids

Pre-built `.cascade` files you can import directly into TCCT to explore the tool.

## How to import

In TCCT, open any pyramid → click **↑ Import** (or use the file picker in a file panel) → select one of these `.cascade` files.

Alternatively, these are full pyramid exports — every block is pre-scored so you can explore the layer view, contradiction map, and synthesis features immediately.

---

## Included examples

### `epistemic_humility_argument.cascade`
**Domain:** Philosophy of mind / epistemology  
**Format:** Argument Map  
**What it shows:** A three-file pyramid (Core Argument → Objections → Synthesis) pressure-testing the claim that epistemic humility is a necessary condition for rationality. Demonstrates:
- Pinned synthesis block with `source_block_ids`
- Genuine tension between Cartesian and Bayesian objections
- A scored synthesis that survives partial refutation

## Creating your own

1. Use a Template (⬡ Templates button on the pyramid list)
2. Add files and blocks
3. Score with AI or set sovereign scores manually
4. Export as `.cascade` (↑ .cascade button in the pyramid view)

The `.cascade` format is plain JSON — you can inspect, version-control, and share these files freely.
